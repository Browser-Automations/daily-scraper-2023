# Dec 01, 2022 - https://www.trulia.com
Script to perform search on trulia.com and extract data from all results  

## How to run
Youtube

## prerequisites

[.NET 7 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/7.0)

## Extracted data fields
* Property Url
* Address
* Images
* Price
* Beds
* Baths
* Area
* Listed By
